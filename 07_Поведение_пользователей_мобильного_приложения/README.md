# Поведение пользователей мобильного приложения

Мы работаем в стартапе, который занимается продажей продуктов питания.

## Этапы исследования

1. Подготовим и изучим данные для анализа. Детально рассмотрим информацию по логам;
2. Изучим вопронку продаж, узнаем какие пользователи доходят до покупки. Сколько доходит до покупки, а сколько — «застревает» на предыдущих шагах;
3. Исследуем результаты A/A/B-эксперимента. Дизайнеры захотели поменять шрифты во всём приложении, а менеджеры сомневаются в данной идее и считают, что пользователям будет непривычно. Управление поставило задачу провести тест, на основании которого будет принято решение по данной инициативе. Пользователей разбили на 3 группы: 2 контрольные со старыми шрифтами и одну экспериментальную — с новыми. Нашей команде предстоит выяснить, какой шрифт лучше;
4. Вывод по результатам A/A/B-эксперимента.

## Задача

Разобраться как ведут себя пользователи нашего мобильного приложения.
