# Увеличение выручки интернет-магазина

### [Проект с полноценной визуализацией и навигацией](https://nbviewer.org/github/BugreevVO/bugreevv_repository/blob/main/05_Проверка_гипотез_по_увличению_выручки_интернет-магазина/bugreev_prioritization_of_hypotheses_to_increase_the_revenue_of_an_online_store.ipynb)

Вместе с отделом маркетинга мы подготовили список гипотез для увеличения выручки интернет-магазина.

## Данные

**Часть 1. Приоретизация гипотез.** В файле `hypothesis.csv` содержится информация о 9 гипотезах по увеличению выручки интернет-магазина с указанными параметрами: `Reach`, `Impact`, `Confidence`, `Effort`.

**Часть 2. Анализ A/B-теста.** Мы провели A/B-тест и получили результаты, которые описаны в файлах `orders.csv` и `visitors.csv`.

## Задача

Приоретизировать гипотезы, запустить А/В-тест и проанализировать результаты.
