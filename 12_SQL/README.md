# Сервис для чтения книг по подписке

### [Проект с полноценной визуализацией и навигацией](https://nbviewer.org/github/BugreevVO/bugreevv_repository/blob/main/12_SQL/bugreev_book_reading_service.ipynb)

## Данные

Из-за пандемии короновируса, жители городов намного реже выходят на улицу, не посещают привычные места (торговые центры, кафе и т.д.), а предпочетают домашний досуг в виде чтения книг. Стартаперы заметили данную тенденцию и бросились создавать приложения для тех, кто любит проводить время за чтением книг.

Наша команда, идет в ногу со временем и новыми обстоятельствами, купила крупный сервис для чтения книг по подписке. 

Работа с базой будет выполнена запросами на языке **SQL**.

## Задача

Проанализировать имеющуюся базу данных, в которой хранится информация о книгах, издательствах, авторах, а также пользовательские рецензии на книги. С помощью этих данных мы сможем сформулировать ценностное предложение для нового продукта.

## Используемые библиотеки

- *pandas*
- *sqlalchemy*
