# Маркетинговый анализ развлекательного приложения

### [Проект с полноценной визуализацией и навигацией](https://nbviewer.org/github/BugreevVO/bugreevv_repository/blob/main/04_Маркетинговый_аналитиз_развлекательного_приложения/bugreev_marketing_analytics_from_an_entertainment_application.ipynb)

Несколько прошлых месяцев наш бизнес постоянно нес убытки - в привлечение пользователей была вложена не малая сумма денег, а результата нет. Нашей команде поставлена задача разобраться в причинах сложившейся ситуации.

## Данные

В нашем распоряжении есть лог сервера с данными о посещениях приложения новыми пользователями, зарегистрировавшимися в период с 2019-05-01 по 2019-10-27, выгрузка их покупок за этот период, а также статистика рекламных расходов. Нам предстоит изучить, как пользователи приложения пользуются продуктом, когда они начинают совершать покупки, сколько денег пиносит каждый клиент, когда он окупается и какие факторы отрицательно влияют на привлечение пользователей.

## Задача

Разобраться в причинах убыточности бизнеса.

## Используемые библиотеки

- *pandas*
- *numpy*
- *datetime*
- *pyplot*
