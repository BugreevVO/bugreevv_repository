# Сегментация пользователей банка

## Данные

В процессе исследования будут определены сегменты пользователей, на основании анализа которых, будут сформулированы и проверены гипотезы. Таким образом, появится возможность интерпретировать результат исследования в рекомендации для дальейшей стратегии развития бизнеса.

*Краткое техническое задание для дашборда*:
1. Постройте диаграмму, отражающую количество клиентов в зависимости от числа используемых продуктов;
2. Добавьте индикатор процента ушедших клиентов;
3. Добавьте фильтр дашборда по городу.

## Задача

Сделать сегментацию пользователей банка по потреблению. Создать дашборд.
